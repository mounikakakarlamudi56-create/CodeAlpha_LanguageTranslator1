# CodeAlpha Language Translator

## 📌 Project Description

This project is a simple web-based Language Translator developed using Python and Flask. It allows users to enter text, choose the source and target languages, and instantly translate the text using a translation library.

## 🚀 Features

* Translate text between multiple languages
* User-friendly web interface
* Fast and accurate translation
* Built with Flask
* Simple and responsive design

## 🛠️ Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* Google Translator Library (`deep-translator` or equivalent)

## ▶️ How to Run the Project

1. Clone the repository:

   ```
   git clone https://github.com/mounikakakarlamudi56-create/CodeAlpha_LanguageTranslator1.git
   ```

2. Navigate to the project folder:

   ```
   cd CodeAlpha_LanguageTranslator1
   ```

3. Install the required packages:

   ```
   pip install -r requirements.txt
   ```

4. Run the Flask application:

   ```
   python app.py
   ```

5. Open your browser and visit:

   ```
   http://127.0.0.1:5000
   ```

## 📸 Output

The application displays a webpage where users can:

* Enter text
* Select source and target languages
* Click the Translate button
* View the translated text instantly
